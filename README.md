# ed2k

Python program that calculates [eD2k hashes] of files and prints them as [ed2k URIs].
These are used for example by [AniDB].

## Usage

```
./ed2k.py FILE [FILE ...]
```

## Dependencies

- Python 3 (tested with 3.11)
- [PyCryptodome]

[eD2k hashes]: https://en.wikipedia.org/wiki/Ed2k_URI_scheme
[ed2k URIs]: https://www.iana.org/assignments/uri-schemes/prov/ed2k
[AniDB]: https://wiki.anidb.net/Ed2k-hash
[PyCryptodome]: https://www.pycryptodome.org/
